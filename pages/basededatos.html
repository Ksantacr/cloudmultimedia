Database

Los servicios de openstack usan una base de datos SQL para almacenar información. Típicamente la base de datos se ejecuta en el nodo controller.

Instalar y configurar la base de datos

  1. Instalar los paquetes: 
    # apt-get install mariadb-server python-mysqldb
  2. Elegir una clave para la cuenta root.
  3. Editar el archivo /etc/mysql/my.cnf y completar las siguientes acciones:
    1. En la sección [mysqld], configurar el parámetro de bind-address con la dirección ip del nodo controller para acceder atravez de la red 'management': 
      [mysqld]
...
bind-address = 10.0.0.11
    2. En la sección [mysqld], configurar los siguientes parámetros: 
      [mysqld]
...
default-storage-engine = innodb
innodb_file_per_table
collation-server = utf8_general_ci
init-connect = 'SET NAMES utf8'
character-set-server = utf8

Finalizar instalación

  1. Reiniciar el servicio de la base de datos: 
    # service mysql restart
  2. Asegurar la base de datos: 
    # mysql_secure_installation
