Acerca del proyecto

Cloud Multimedia es un proyecto que ayuda a estudiantes de Espol, reduciendo tiempos de espera al realizar un renderizado con 3ds max 2016 y aprovechar al máximo los recursos de cómputo, almacenamiento y redes.

Openstack

Para el proyecto se utilizó la versión Juno-Openstack. Openstack se divide en módulos que se encargan de tareas específicas, Los componentes mas importantes son:

  • Compute (Nova)

    Se encarga del ciclo de vida de las instancias en el entorno Openstack y es responsable del intercambio, programación y eliminación de las máquinas virtuales.

  • Networking (Neutron)

    Ofrece una API a los usuarios para crear redes y enlazarlas.

  • Dashboard (Horizon)

    Provee una interfaz web para controlar los servicios de Openstack, cómo, lanzar instancias, asignación de redes y configuración del control de acceso.

  • Identity service (Keystone)

    Proporciona la autenticación y autoriza el acceso a los diferentes servicios de Openstack.

  • Image service (Glance)

    Almacena y guarda las imágenes de las máquinas virtuales, para poder instarciarlas.
